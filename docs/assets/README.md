# MedTrust Logo Assets

## Logo Files

- **logo.svg** - Main logo (vector, scalable)
- **logo-text.svg** - Logo with text
- **logo-icon.svg** - Icon only (for app)

## Logo Concept

The MedTrust logo combines three key elements:

1. **Medical Cross** (White) - Represents healthcare and medical care
2. **AI Circuit Pattern** (Green) - Represents artificial intelligence and technology
3. **Blue Circle** - Represents trust, reliability, and global reach
4. **Pulse Line** (Orange) - Represents life, vitality, and real-time monitoring

## Color Palette

- **Primary Blue**: #0066CC - Trust, healthcare, professionalism
- **Trust Green**: #00A86B - Growth, health, AI technology
- **Accent Orange**: #FF6B35 - Energy, warmth, accessibility
- **White**: #FFFFFF - Clarity, simplicity, purity

## Usage Guidelines

### Do's
✅ Use on white or light backgrounds
✅ Maintain minimum size of 40px height
✅ Keep adequate spacing around logo
✅ Use official color palette

### Don'ts
❌ Don't distort or stretch the logo
❌ Don't change colors
❌ Don't add effects or shadows
❌ Don't place on busy backgrounds

## Logo Variations

### Primary Logo
Use for main branding, website header, app icon

### Logo with Text
Use for presentations, documents, marketing materials

### Icon Only
Use for app icon, favicon, social media profile

## File Formats

- **SVG**: For web, scalable applications
- **PNG**: For presentations, documents (high resolution)
- **ICO**: For favicon

## Generating PNG from SVG

To convert SVG to PNG, you can use:

```bash
# Using ImageMagick
convert -background none logo.svg -resize 512x512 logo.png

# Using Inkscape
inkscape logo.svg --export-png=logo.png --export-width=512

# Online tools
# https://cloudconvert.com/svg-to-png
# https://svgtopng.com/
```
